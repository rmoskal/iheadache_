Setting iHeadache Preferences

(My Profile Tab > Acute Treatments)

iHeadache uses rules inspired by the International Headache Society to determine headache type (Migraine, Probable Migraine, and Tension types are currently supported). The “My Profile” area is used to set application “preferences” that are used for headache classification and reports. Want to learn more about headache classifications or types? Click here to read about the International Headache Society headache classification.

Headache Order Test Preference

It is possible for a single headache to meet criteria for both Tension Type Headache and Probable Migraine Headache.  For those headaches that meet both criteria, the system will report the one the one that you specify here.

Headache Duration Rules

The diagnosis of migraine headache requires an individual headache to last at least 4 hours (or less than 4 hours if an acute headache specific medication is taken).  The diagnosis of tension headache requires an individual headache to last at least 30 minutes.  Since many users forget to put in headache duration the following options can be set.  

  • Use duration if entered  - We recommend using this setting.  The system will use the time criteria when it has been entered and ignore time criteria when it is not entered and not needed to classify the headache type.  If the system can determine the headache type based upon the data entered, it will display the headache type as tension, migraine, or probable migraine. If there is not enough info about the headache to determine the type then the type will be set to “Unclassified Headache”.
  • Ignore Duration - The headache duration will be ignored and not used in any of the headache classification analysis.
  • Require Duration – The time requirements will be required for classification of all headaches.  Therefore, all headaches without a stop time will be classified as “Unclassified Headache”

Display Medication Names in Reports

You may use trade names, generic names, or both generic and trade names in reports.

  • Trade (Brand) name examples are:  Imitrex®, Topamax®, Advil®
  • Grand name examples are:  Sumatriptan, Topiramate, Ibuprofen
