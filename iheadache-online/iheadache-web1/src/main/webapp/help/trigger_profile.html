Tracking your Triggers

(My Profile Tab >Trigger Profile)

Use this page to set the possible headache triggers you desire to track. This way you only see the triggers that you want to track instead our entire list of over 40 triggers.

Common [Trigger] Tab 

The common trigger list has headache triggers that are most commonly identified by physicians and research. Check off the triggers you wish to track.  If you decide that you don’t want to track a particular trigger anymore, then just uncheck it.

Custom [Trigger] Tab

Triggers that are not in the common list can be added to our “Custom Tab” This allows you to track as many triggers as desired.  Click Add and enter the desired trigger.  You must hit the update button and the OK button to save your entry.

  • To stop tracking the custom trigger but allow yourself to track it later, unclick the “track” box. 
  • If you no longer wish to ever track a custom trigger, you can select a trigger and hit the delete button but this is not recommended.  It is better to just “uncheck” it and not include it in your Trigger Profile.
